05 / THE VERDICT: HIGH-LEVEL ARCHITECTURAL LESSONS
1. Context Lifecycle Management LLM-based agents experience cognitive decay at 100% context saturation. Systems must enforce background summarization ticks and hot-swapping agent instances for long outages.
2. Tenant Separation & Boundaries SRE agents must be strictly isolated per-tenant. An agent spanning multiple environments represents an unacceptable security blast radius for enterprise infrastructure.
3. Identity & RBAC Boundaries Bespoke per-service identities are required. Shared credentials prevent defensive agents from applying IAM-level blocking or restricting Kubernetes permissions during attack loops.
4. Automation Sweep Cadence Manual troubleshooting speed is useless against automated attackers. Defense systems require pre-deployed 10-second sweep loops to continually undo rogue infrastructure mutations.
